# Virale — лендинг (для размещения на chatplace.io/ru/virale)

Статический лендинг. **Без сборки, без бэкенда, без базы данных.** Один HTML-файл + папка с медиа и шрифтами. Достаточно отдать содержимое этой папки любым веб-сервером.

## Что внутри
- `index.html` — вся страница (HTML + CSS + JS в одном файле, ванильный JS, без фреймворков).
- `fonts/` — шрифты Pragmatica Extended и Roboto (self-hosted, `.ttf`).
- `*.jpg / *.webp / *.png / *.mp4` — изображения и видео (демо-ролики, обложки, фото).

Всего ~60 файлов, ~30 МБ (основной вес — демо-видео `.mp4`).

## Как развернуть (выбрать любой способ)

### Вариант A. Просто положить файлы на сервер
1. Залить **всё содержимое этой папки целиком** (вместе с `fonts/`) в каталог, который отдаётся по адресу `https://chatplace.io/ru/virale/`.
2. Убедиться, что сервер по умолчанию отдаёт `index.html`.
3. Открыть `https://chatplace.io/ru/virale/` — всё должно работать.

> ВАЖНО: сохранить структуру папок — `fonts/` должна лежать рядом с `index.html`. Все ссылки внутри относительные, никакие пути в коде менять не нужно.

### Вариант B. Nginx (пример конфига)
```nginx
location /ru/virale/ {
    alias /var/www/virale-landing/;   # путь, куда залили эту папку
    index index.html;
    try_files $uri $uri/ /ru/virale/index.html;
}
```

### Вариант C. S3 / CDN / любой статик-хостинг
Загрузить папку как есть, точка входа — `index.html`. Работает на S3+CloudFront, Cloudflare Pages, Netlify, GitHub Pages и т.п.

## Требования к серверу
- Любой статический веб-сервер (nginx, Apache, S3/CDN). PHP/Node/БД не нужны.
- Корректные MIME-типы (обычно по умолчанию уже так):
  - `.mp4` → `video/mp4`
  - `.webp` → `image/webp`
  - `.ttf` → `font/ttf`
- Рекомендуется HTTPS (видео `autoplay` и шрифты).

## Внешние зависимости (грузятся с CDN, не входят в папку)
- **Иконки Tabler** — `https://cdn.jsdelivr.net/npm/@tabler/icons-webfont@3/dist/tabler-icons.min.css`.
  Работает «из коробки». Если нужна полная автономность/стабильность — можно self-host: скачать этот CSS и шрифты пакета `@tabler/icons-webfont` и заменить ссылку в `<head>` на локальную.
- **og:image** для превью в соцсетях указывает на `https://storage.googleapis.com/chatplace-website/meta/ru/virale.png` (уже на вашей инфраструктуре). При необходимости поменять путь в мета-тегах `<head>`.

## Проверка после деплоя
- Страница открывается, шрифты подгрузились (заголовки — Pragmatica Extended).
- Демо-видео в макете телефона и в видеостудии проигрываются и зациклены.
- На мобильном: нет горизонтальной прокрутки, блоки листаются без рывков.
- В консоли браузера нет 404 на ассеты (значит структура папок сохранена).

## Правки текста/контента
Весь контент — прямо в `index.html` (обычный текст в разметке). CSS и JS — в `<style>` и `<script>` внутри того же файла.
